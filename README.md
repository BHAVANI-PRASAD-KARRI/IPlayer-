# 🎵 IPlayer

IPlayer is a simple and modern music player application designed to play audio files with an easy-to-use interface and smooth user experience.

---

## ✨ Features

- 🎧 Play and pause songs
- ⏭️ Next and previous track controls
- 🔊 Volume control
- 📃 Playlist support
- ⏱️ Song progress bar
- 🌙 Clean and responsive UI
- 📱 Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
